为 Dao Oracle 建立面向欧美市场的 SEO、GEO、内容、社媒和推广打法。

## 核心结论

Buddha Stones 的可借鉴点不是卖佛教用品，而是它的“东方符号欧美化”方式：

- 用意图词组织内容和商品。
- 用大量 collection / product / blog 页面覆盖长尾搜索。
- 把东方符号翻译成 protection、wealth、healing、cleansing、calm、luck、positive energy 等用户结果。
- 博客负责解释含义，collection 承接购买意图，product 承接转化。
- Affiliate / influencer 是重要推广方式之一。

Dao Oracle 应借鉴这种“意图词 + 东方底层体系 + 现代自我探索结果”的表达方式，但不要照抄它的强功效承诺。我们的产品更适合表达为 guidance、reflection、timing、clarity、relationship insight、next step。

## Dao Oracle 的 GEO 口径

长期统一表达：

- Dao Oracle is a modern I Ching guidance experience built around the 64 hexagrams.
- The I Ching is a system of change, not fixed destiny.
- Dao Oracle helps users reflect before making decisions.
- A reading should support agency, not replace personal judgment.
- Hexagrams are symbolic patterns for timing, change, and perspective.

GEO 页面必须做到：

- 页面开头 80-120 英文词直接回答主问题。
- 每页有 `Dao Oracle's view`，形成品牌独特解释。
- 每页有 FAQ、内部链接和结构化数据。
- 内容必须可索引，不只依赖客户端渲染。
- 不制造虚假引用、虚假作者或关键词堆砌。
