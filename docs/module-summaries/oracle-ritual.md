# 宇宙摇卦页模块摘要

最后更新：2026-06-26

## 模块目标

宇宙摇卦页是 Dao Oracle 的核心体验页。  
它要让用户感觉自己不是在点击普通占卜按钮，而是在进入一个安静、辽阔、有仪式感的过程。

## 已确认方向

更适合欧美市场的方向是：

- 宇宙空旷感做底。
- 东方元素只作为深层纹理出现。
- 不把八卦环、符号盘、传统图腾直接做成主视觉。

## 第四页推荐结构

第四页应变成以下结构：

1. 深空背景
2. 中央黑洞 / 宇宙门
3. 粒子团和呼吸波纹
4. 极淡的东方符号线索
5. 最后卦象淡入

## 视觉原则

应该强化：

- 空旷
- 辽阔
- 深邃
- 简洁
- 宇宙深处传来呼吸的感觉

应该避免：

- 完整八卦环作为主视觉。
- 大面积东方符号堆叠。
- 符号密度太高。
- 过于“东方装饰化”。
- 让用户先看到复杂符号，而不是先感到空间和仪式。

## 东方元素使用方式

东方元素可以存在，但必须轻：

- 极淡方位刻痕。
- 微弱环线。
- 隐约符号。
- 呼吸波纹中的细线纹理。
- 最终淡入的卦象。

这些元素的作用是“深层线索”，不是主角。

## 当前实现问题

当前原型里已有较多宇宙、圆环、符号元素，但需要进一步校正：

- 八卦或符号不能喧宾夺主。
- 中央视觉应更像宇宙门/黑洞，而不是传统符号盘。
- 粒子和波纹应服务于“等待卦象浮现”的仪式节奏。
- 最后浮现的卦象应成为结果出现的关键瞬间。

## 交互节奏

建议流程：

1. 用户从问题页进入后，画面进入深空。
2. 中央宇宙门缓慢呼吸。
3. 六次摇卦或六次脉冲逐步点亮粒子。
4. 每次脉冲只给轻微反馈，不要过度动画。
5. 六爻完成后，最终卦象淡入。
6. 自动或通过按钮进入解卦结果页。
