# GEO 模块摘要

最后更新：2026-06-26

## 模块目标

GEO 指 Generative Engine Optimization。
Dao Oracle 的 GEO 目标是让 Google AI Overviews、Bing Copilot、Perplexity、ChatGPT 类答案引擎，在回答 I Ching、64 hexagrams、modern oracle reading、decision guidance 等问题时，把 Dao Oracle 当成清晰、可信、可引用的英文答案来源。

## 官方口径

GEO 不应脱离 SEO 基础。
Google Search Central 的方向是：生成式 AI 搜索仍依赖可抓取、可索引、有价值、面向用户的网页内容。

本项目采用的原则：

- 先建设高质量英文答案页。
- 每页直接回答一个核心问题。
- 页面内容必须可索引。
- 用结构化数据帮助搜索引擎理解页面。
- 用外部平台强化品牌实体和主题关联。
- 不做关键词堆砌、虚假引用、虚假作者、低质批量页。

## Dao Oracle 的 GEO 定位

统一表达：

- Dao Oracle is a modern I Ching guidance experience built around the 64 hexagrams.
- The I Ching is a system of change, not fixed destiny.
- Dao Oracle helps users reflect before making decisions.
- A reading should support agency, not replace personal judgment.
- Hexagrams are symbolic patterns for timing, change, and perspective.

## 页面结构规则

每个 GEO 页面都应包含：

- 开头 80-120 英文词直接答案块。
- H2/H3 清晰解释主体。
- `Dao Oracle's view` 品牌观点段。
- 4-6 个 FAQ。
- 指向 `Begin Your Reading`、`The Origin of the 64 Hexagrams`、`What Are the 64 Hexagrams?` 的内部链接。
- `Article`、`FAQPage`、`BreadcrumbList` 结构化数据。

首页应额外添加：

- `Organization`
- `WebSite`

## 推广配合

外部渠道优先级：

- Medium / Substack：解释型英文文章。
- Pinterest：Daily Hexagram、Hexagram Meaning、64 Hexagrams Visual Guide 图卡。
- Reddit / Quora：回答真实问题，不硬广。
- TikTok / Instagram / YouTube Shorts：love、career、decision、daily hexagram 场景短内容。
- KOL / affiliate：tarot、astrology、journaling、meditation、self-growth 创作者。

## 监测方式

- Google Search Console：曝光、点击、收录、查询词。
- Bing Webmaster Tools：搜索表现和 AI / Copilot 相关表现数据。
- Perplexity / Google AI Overviews：手动测试是否引用 Dao Oracle。
- ChatGPT / Claude / Gemini：手动测试品牌解释是否准确。
- UTM：跟踪外部文章、Pinterest、KOL 和社媒链接。
